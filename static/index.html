Welcome to Cloud Is Us, the platform for processing large graph datasets in a social cloud computing setting. In fact, it's a bit like SETI@home; remember that one? Tune in now to donate a tiny fraction of your machine power for the greater good. No matter if you have a laptop, desktop, smart phone, or an iPad or Android tablet — all you need is a Web browser and you can join in and crunch some data. Actually, since you're reading this you likely have everything you need to get started ...

So, start contributing now or learn more about how Cloud Is Us works.
